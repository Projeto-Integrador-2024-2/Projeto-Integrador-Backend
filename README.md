# Projeto-Integrador-Frontend

### Como Usar 

Para usar o Backend teste instale django:
    ```pip install -r requirements.txt```
e
    ```py .\manage.py runserver```


Para abrir o Frontend 
    ```cd .\frontend\```
e
    ```npm install```
por fim
    ```npm run dev```

Para fazer migration dos modulos
    ```./manage.py makemigrations```
e
    ```./manage.py migrate```
